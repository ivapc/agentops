# agentops

[![CI](https://github.com/ivanrdvc/agentops/actions/workflows/ci.yml/badge.svg)](https://github.com/ivanrdvc/agentops/actions/workflows/ci.yml)
[![Release](https://img.shields.io/github/v/release/ivanrdvc/agentops?include_prereleases&sort=semver)](https://github.com/ivanrdvc/agentops/releases)

Observability and monitoring for AI agents — traces, runs, evals, and MCP activity in one place.

Requires Node 22 and pnpm 10. Run `pnpm install && pnpm dev` and open <http://localhost:3000>. See [`docs/`](docs/README.md) for architecture.
